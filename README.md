# Hudley

Visualize your car's data and overlay it on your video


## How to dev

```
npm i
npm start
```

## How to test

```
npm test
```

## How to lint

```
npm run lint
```

and to fix all auto-fixable linting issues

```
npm run lint -- --fix
```


## Project Structure

```
```
- src/
  - main/     node code
  - renderer/ vue code
  - shared/   code shared between node and vue
  - preload/  expose node code to vue code
